# 0x03. Shell, init files, variables and expansions

This project covers shell variables, expansions, and command execution in Linux. The tasks focus on creating aliases, working with environment variables, performing arithmetic operations, and understanding how the shell interprets commands.

## Files

- **0-alias**: Creates an alias named `ls` with the value `rm *`.
- **1-hello_you**: Prints “hello” followed by the current Linux user.
- **2-path**: Adds `/action` to the end of the existing PATH.
- **3-paths**: Counts the number of directories in the PATH.
- **4-global_variables**: Prints all global environment variables.
- **5-local_variables**: Lists all local variables, environment variables, and shell functions.
- **6-create_local_variable**: Creates a local variable named `BEST` with the value `School`.
- **7-create_global_variable**: Creates a global variable named `BEST` with the value `School`.
- **8-true_knowledge**: Prints the result of `TRUEKNOWLEDGE + 128`.
- **9-divide_and_rule**: Prints the result of `POWER / DIVIDE`.
- **10-love_exponent_breath**: Prints the result of `BREATH` raised to the power of `LOVE`.

## Requirements

- All scripts start with `#!/bin/bash`
- All scripts are executable
- Each script contains exactly two lines
